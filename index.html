KỸ NĂNG CHUYÊN MÔN
  HTML VÀ CSS
  • Biết tư duy phân tích thiết kế layout.
  • Nắm được các bộ thẻ cốt lõi để xây dựng khung HTML.
  • Các bộ chọn trong css như: selector id, selector css, selector con cháu, selector before, after...
  • Các thuộc tính cho text, thuộc tính định dạng cho khối.
  • Các hiệu ứng như: làm trong suốt khối với opacity, làm mềm quá trình thay đổi css với transition, Xoay khối với Rotate...
  • Biết bố cục trang với Flexbox.
  • Quy tắc đặt tên chuyên nghiệp cho class, id.
  • Các plugin quan trọng như: thêm google font, fontawesome, slider với owl carousel, nhúng các tiện ích của facebook.
  • Biết cách sử dụng Photoshop cơ bản để chuyển PSD thành HTML CSS.

  BOOTSTRAP

  • Xây dựng layout website responsive với Grid System.
  • Nắm được các class cơ bản để xây dụng giao diện nhanh hơn, sử dụng được các cấu trúc để xây dựng khung trong HTML như: Navbar menu responsive, Carousel slideshow, Các loại Form, các loại Card trong bootstrap...

  JQUERY

  • Các nhóm selector
  • Các phương thức xử lí trong jquery: thuộc tính, class, giá trị tìm kiếm lọc...

  • Sự kiện trong jquery

  PHP

  • Nắm được dữ liệu trong php, những phép toán trong php, cấu trúc điều khiển và vòng lặp.
  • Hiểu rõ tầm quan trọng khi viết hàm và cách đặt tên hàm rõ ràng dễ hiểu.
  • Nắm được các kiểu dữ liệu truyền lên server: POST và GET.
  • Biết chuẩn hóa dữ liệu form.
  • Hiểu phiên làm việc: COOKIE, SESSION.
  • Nắm được các thư viện hàm.
  • Biết làm việc giữa PHP với Ajax.
  • Biết tạo đường link thân thiện cho website.
